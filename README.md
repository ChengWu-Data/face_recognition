Face Detection and Face Recognition Project
===========================================

Student GitHub Project Files
----------------------------
This project was written to match the assignment requirements:

1. Use the provided Haar cascade file `haarcascade_frontalface_default.xml`
   to detect and crop faces from the original Ariel Sharon images.
2. Save the cropped Ariel Sharon face images into a folder named
   `ariel sharon by hands`.
3. Train a face recognition model on six classes:
   - ariel_sharon
   - chris evans
   - chris hemsworth
   - mark ruffalo
   - robert_downey_jr
   - scarlett_johansson
4. Split the dataset with:
   - `test_size = 0.3`
   - `random_state = 42`
5. Use training data only for model fitting.
6. Use the Eigenfaces approach for face recognition.
7. Print the final recognition accuracy.

Code Logic
----------
The program runs in two main parts.

Part 1: Face Detection
----------------------
1. The script first checks whether the `Faces` folder and the XML cascade file exist.
2. It looks for the original Ariel Sharon image folder.
3. It uses OpenCV's Haar cascade detector from `haarcascade_frontalface_default.xml`
   to detect faces in each Ariel Sharon image.
4. If more than one face is found, it keeps the largest one.
5. The detected face is cropped, converted to grayscale, resized to `100 x 100`,
   and saved into the folder `ariel sharon by hands`.
6. The saved filename style is kept consistent with the rest of the dataset as much as possible.

Part 2: Face Recognition
------------------------
1. The script loads the six required classes.
2. Every image is read in grayscale and resized to a common size.
3. Every face image is flattened into a one-dimensional vector.
4. The dataset is split into training and testing sets using:
   - `test_size = 0.3`
   - `random_state = 42`
5. The training data is standardized.
6. PCA is applied on the training data only.
   This PCA representation is the Eigenfaces step.
7. A KNN classifier is trained on the Eigenfaces features from the training set only.
8. The model predicts the labels of the test set.
9. The final accuracy is printed on the screen.

Problems Found During the Assignment
------------------------------------
1. Folder naming may not always be fully consistent
   Some datasets use spaces, some use underscores, and some use uppercase folder names.
   To make the script more stable, I added a normalization step when matching folder names.

2. Ariel Sharon source images may contain no face or multiple faces
   Haar cascade is simple and fast, but it is not perfect.
   In some difficult images, it may fail to detect the face or detect more than one region.

3. Image filename patterns may vary
   Since the assignment asks the Ariel Sharon cropped files to follow the same naming format
   as the other classes, I wrote the script to inspect an existing folder and imitate that pattern.
   If the original dataset naming style changes, this part may still need a small manual check.

4. Accuracy can change depending on the dataset quality
   The final recognition accuracy depends on image quality, lighting, pose, background,
   and how clean the cropped faces are. Even with the same code, results may change if the
   dataset content changes.

Possible Improvements
---------------------
1. Use a stronger face detector
   Haar cascade is required for this assignment, but modern detectors such as MTCNN or Dlib
   are usually more accurate for difficult images.

2. Add face alignment
   If eyes and face orientation are aligned before recognition, Eigenfaces usually performs better.

3. Tune the classifier and PCA dimension
   The current code uses KNN after PCA. Accuracy may improve by testing:
   - different `n_components`
   - different KNN neighbor counts
   - SVM instead of KNN

4. Improve image cleaning
   Better cropped faces, more balanced data, and more consistent lighting would likely improve
   recognition accuracy.

How to Run
----------
1. Open a terminal in the project folder.
2. Install the required packages:
   `pip install -r requirements.txt`
3. Run the script:
   `python main.py`

Output
------
The script will print:
- how many Ariel Sharon cropped face images were saved
- the final test accuracy
- the number of training samples
- the number of testing samples
- the number of Eigenfaces used

Notes
-----
- This script does not use absolute paths.
- The `.py` file should stay at the same directory level as the `Faces` folder.
- The recognition model is trained strictly on the training dataset only.
- The test set is used only for evaluation.
